1-¿Qué atributo se utiliza para especificar la URL a la que se enviarán los datos del formulario? respuesta: (b)-action

2-¿Cuál es el atributo que se utiliza para especificar el tipo de datos que se espera en un campo de entrada? respuesta: (c)-type

3-¿Cuál de los siguientes tipos de campo de entrada se utiliza para ingresar direcciones de correo electrónico? respuesta: (b)-email

4-¿Cuál es el elemento HTML que se utiliza para asociar un texto descriptivo con un campo de entrada en un formulario? respuesta: (d)-<label>

5-¿Cuál de las siguientes opciones es la etiqueta correcta para crear un botón de envío en un formulario? respuesta: (a)-<input type="submit">

6-¿Cuál de los siguientes atributos se utiliza para hacer que un campo de entrada sea obligatorio en un formulario? respuesta: (a)-required

7-¿Cuál de los siguientes elementos HTML se utiliza para crear una lista desplegable en un formulario? respuesta: (c)-validate

8-¿Cuál de los siguientes tipos de campo de entrada se utiliza para ingresar un apellido? respuesta: (a)-text

9-¿Qué atributo se utiliza para definir un valor predeterminado en un campo de entrada de un formulario? respuesta: (a)-value

10-¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos. Bibliografía MDN

-get: Solicita datos del servidor, normalmente para recuperar información sin alterar su estado.
-post: Envia datos al servidor, comúnmente para crear o actualizar un recurso.
-put: Reemplaza un recurso existente en el servidor.
-delete: Elimina un recurso específico en el servidor.

11-¿Qué son los códigos de estado de respuestas HTTP? Realizar una tabla con su clasificación. Bibliografía MDN

|codigo| |categoria| |descripcion|
|1xx| |informativo| |la solicitud fue recibida y el proceso sigue|
|2xx| |exito| |la solicitud se proceso con exito|
|3xx| |redireccion| |se requiere alguna accion adicional para complementar la solicitud|
|4xx| |error del cliente| |la solicitud tiene un error o no se puede procesar|
|5xx| |error del servidor| |el servidor fallo al completar la solicitud|
